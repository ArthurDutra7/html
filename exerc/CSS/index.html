Trabalhando fontes CSS

Tags para Fontes

Font-size, p/colocar o tamanho das fontes

Font-weight, tag peso da fonte.

Font-style, tag p/escolher como quer a fonte (italico,etc.)

Text-decoration, tag para escolher o que colocar na fonte (sublinhado,etc.)

Para ocupar menos espaço, usamos o shorthand;
(font style -> font weight -> font size -> font family).
Ex: font: italic bolder 17px Arial, sans-serif

Site google fonts, p/encontrar fontes.
Para adicionar basta usar a tag (@import), não pode estar dentro de um seletor.
Ex: body.

Site Dafont, p/pegar fontes externas.
E usar a tag (@font-face)

Extensões baixadas para descobrir fontes, cores, etc. usadas em algum site.

Site (Whatfontis), p/achar fontes dentro de imagens.

Tag (text-indent), faz criar o espaço no começo do parágrafo.

Tag (Text-align), p/centralizar aonde o texto vai ficar

PAra mudar apenas um H1 tendo diversos, basta usarmos a tag (id) e no CSS usarmos a tag (#, e o nome dado).

Usamos a tag class p/modificar os elementos da mesma classe, basta usar a tag (class e o nome) no HTML e no CSS basta apenas colocar um (., e o nome).

O (id) não pode ser usada + de 1 vez na mesma pág. html, mas pode ser usado na mesma classe um id e uma class.

O (class) pode ser usado + de 1 vez, basta colocar (class -> 1nome -> espaço -> 2nome...).

tag (Div : hover), p/ter animações ao passar o mouse em cima.
Usamos (>) p/expecificar em qual div queremos mudar.
